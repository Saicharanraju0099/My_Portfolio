## 🚀 Running Locally

To run a local copy of this project on your machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Saicharanraju0099/My_Portfolio.git
   ```

2. **Navigate into the directory:**
   ```bash
   cd My_Portfolio
   ```

3. **Install the dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **View the site:** Open your browser and navigate to `http://localhost:5173/`
